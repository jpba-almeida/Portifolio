## Como usar

1. Clone o repositório:

```bash
git clone https://github.com/saadpasta/developerFolio.git
cd developerFolio
```

2. Instale dependências:

```bash
npm install
```

3. Configure variáveis de ambiente:

```env
REACT_APP_GITHUB_TOKEN="SEU_TOKEN_GITHUB"
GITHUB_USERNAME="SEU_USUARIO_GITHUB"
USE_GITHUB_DATA="true"
```

4. Rode localmente:

```bash
npm start
```

5. Customize `src/portfolio.js` e cores em `src/_globalColor.scss`.

## Seções incluídas

* Resumo / Sobre mim
* Skills, Educação, Experiência
* Projetos Open Source e Big Projects
* Certificações, Blogs, Talks, Podcasts
* Contato, Twitter, GitHub Profile
